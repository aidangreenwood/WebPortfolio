Python Projects

Throughout CSE 130, I completed several Python projects that strengthened my understanding of programming fundamentals, algorithms, input validation, and modular program design. Below are some of my favorite projects from the course.

  • Lab 01: Guess the Number

    A number guessing game where the user tries to guess a randomly generated number while receiving hints after each guess.

  • Lab 02: Authentication

    A login verification program that reads usernames and passwords from a JSON file and checks whether the user is authorized.

  • Lab 04: Monopoly

    A decision-based program that determines whether a player can purchase a hotel on Pennsylvania Avenue based on Monopoly rules, available houses, hotels, and cash.

After this point, the course projects began using a two-part process: one lab focused on planning and pseudocode, and the following lab focused on implementing that plan in Python. The planning documents are linked with the Python projects they helped prepare.

  • Lab 06: Image Compression

    Reads compressed image data from a JSON file and reconstructs the image as ASCII art in the console. This project introduced file processing and working with structured data. Planning (PDF)

  • Lab 08: Selection Sort

    Implements the selection sort algorithm to organize values stored in a JSON file while emphasizing assertions, modular functions, and algorithm analysis. Planning (PDF)

  • Lab 10: Number of Days Calculator

    Calculates the number of days between two dates while accounting for leap years, invalid input, and calendar edge cases through extensive validation. Planning (PDF)
